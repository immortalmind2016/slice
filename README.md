# How to run, without docker

- postgres must be installed in your machine `We will discuss how to use docker instead`
- npm install
- npm run start:dev
- open browser <a href="http://localhost:3000">http://localhost:3000</a>

# With docker `recommended`

- docker-compose up -d
- open browser <a href="http://localhost:3000">http://localhost:3000</a>
